## 6. 기술적 하이라이트

### 6.1 다변수 최적화 (채권 트레이더 방법론 이식)
```
Score = 0.30·popdens + 0.30·lst + 0.20·vuln − 0.15·shade − 0.05·natural
          (모든 피처 MinMax 정규화 후 가중합)
```

### 6.2 시간대 가중 스코어링
- 9~18시 10개 시간대별 독립 필드 생성
- HOUR_WEIGHTS 로 오후 1~3시 피크 강조 (1.0 가중)
- 역세권(출퇴근)과 시장·상업(오후) 프로파일 분리

### 6.3 보행로·횡단보도 강제 제약
- 3,672 격자 → **23 격자** (0.6%)로 축소
- 강변·건물 위 등 설치 불가능 지점 원천 제외
- 실제 정책 결정자가 바로 활용 가능한 입지만 남김

### 6.4 시나리오 민감도 분석
- 동일 피처 × 4개 가중치 프리셋 → 강건 입지 식별
- 정책 관점 불변 입지 (robust location) 3곳 확인

### 6.5 LLM 자동 근거 생성
- TOP 후보지 메타데이터 → GPT API → 정책 결정자용 자연어 설치 근거
- (API 키 없으면 룰베이스 템플릿 fallback)

### 6.6 자연 그늘 시뮬레이션 (P1 완료)
- 동작구 주요 건물군 footprint(폭염 피크 지점 19개) + 오후 3시 태양위치
  (고도 52° / 방위 252°, 7월 말 서울 기준)
- 각 건물의 그림자 polygon = 건물 footprint ∪ 태양반대방향 평행이동의 convex hull
- 격자별 자연그늘 커버 비율 [0,1] → `natural` 음수 가중 페널티
- 효과: 강건 입지 4 → 3, 상도로 score 0.395 → 0.382 (그림자 영향 반영)

### 6.7 인터랙티브 대시보드 (P3 완료)
- Streamlit (`streamlit run app.py`)
- 사이드바: 시나리오 프리셋 + 5개 가중치 슬라이더 (popdens·lst·vuln·shade·natural)
- 본문: TOP 10 지도(Folium 임베드) + 표 + 가중치 합계/통계
- 캐싱: `@st.cache_resource`로 격자·기본 피처 1회 계산, 슬라이더는 가중합 재계산만
